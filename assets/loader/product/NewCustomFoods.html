プライバシーポリシー

概要

Spigot 1.8 ~ 1.12 向けのプラグインです。
独自の食料システムを追加します。
体力値回復量・食料値回復量・クールダウン・サウンド・パーティクル・ポーション効果・コマンド実行・食料無限化を各食糧ごとに設定できます。

コマンド

コマンド 説明
/cf create [ID] 食料を作成します。
/cf edit [ID] IDの食糧を編集します。
/cf give [ID] [amount] [player] 対象プレイヤーにIDの食料を対象数与えます。
/cf list コンフィグに記載されている食料を一覧表示します。
/cf save 現在の食糧設定をコンフィグに保存します。
/cf reload コンフィグをリロードします。
/cf help ヘルプを表示します。

パーミッション

パーミッション 説明
CustomFoods.op コマンドを実行できます。
