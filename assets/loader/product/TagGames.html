プライバシーポリシー

概要

Spigot 1.8 ~ 1.12 向けのプラグインです。
4種類のルールの鬼ごっこを追加します。
鬼のランダム指定・終了時間設定などができます。

コマンド

コマンド 説明
/tag mode [change/infection/ice/height] ゲームの種類設定を行います。
/tag time [second] ゲーム時間を設定します。
/tag start ゲームを開始します。
/tag stop ゲームを強制停止します。
/tag join 自身をゲームに参加させます。
/tag join [player/*] 指定プレイヤーをゲームに参加させます。
/tag leave 自身をゲームから離脱させます。
/tag leave [player/*] 指定プレイヤーをゲームから離脱させます。
/tag tag add ランダムで鬼を一人設定します。
/tag tag add [player] 指定プレイヤーを鬼に設定します。
/tag tag remove ランダムで鬼から一人除外します。
/tag tag remove [player] 指定プレイヤーを鬼から除外します。
/tag list ゲーム参加者を一覧表示します。

パーミッション

パーミッション 説明
TagGames.op ゲーム管理系コマンドを実行できます。
